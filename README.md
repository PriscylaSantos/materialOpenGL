## OpenGL  

### Instalação OpenGL
 
#### Linux
Atualize os pacotes  

    $ sudo apt-get update
    $ sudo apt-get upgrade  

Instale 

    $ sudo apt-get install gcc  
    $ sudo apt-get install freeglut3-dev
    $ sudo apt-get install build-essential

Compilando um arquivo

    $ gcc arquivo.c -o arquivo -lglut -lGL -lGLU -lm

Executando  

    ./arquivo
